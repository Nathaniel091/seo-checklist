My SEO checkList

  1. Favicon - (Done with the 'Link tag')
  2. Title tag
  3. META charset - (UTF-8)
  4. META Viewport - (for website scaling)
  5. META Author (Website author - important especially for blog websites)
  6. META description - (short description of the page content which will include keywords)
  7. META keywords
