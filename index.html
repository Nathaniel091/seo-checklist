8 most important SEO tags

  1. Title tag - <title>some text here</title>

  2. "Responsive" design meta tag - <meta name="viewport" content="width=device-width, initial-scale=1.0">

  3. META description tag - (short description of the page's content which must include keywords too) <meta name="description" content="[some text here]">

  4. Canonical link tag - <link rel="canonical" content="[some url here]">

  5. "Alt" text tag - <img alt="[some text here]">

  6. "Robots" meta tag - <meta name="robots" content="[some parameters here]">
  7. "Open graph" meta tags & "Twitter cards"

  8. "Header" tags - <h1> </h1>

Other important tags

  1. META charset tag - <meta charset="UTF-8">

  2. META keywords tag - (a list of keywords matching the page's content. Keyword must appear in title, meta description, heading, sub-heading, opening paragraph & anywhere possible) <meta name="keywords" content="[parameter1, parameter2, parameter3, ...]">

  3. META author tag - <meta name="author" content="[author's name here]">

  4. icons (2) link tag - ("link" tag with attribute "rel"=canonical) <link href="favicon.jpg" rel="icon">
    5. <link href="apple-touch-icon.jpg" rel="icon">

My SEO & Optimization Checklist

  1. Add all the important SEO tags

  2. Favicon - (add an icon with the 'Link tag')

  3. Add 'Alt' attribute for all images tags

  4. Add 'title' attribute for 'anchor' tags

  5. Mobile friendliness: mobile phone traffick is on the increase

  6. Compress images(JPEG) for faster loading [e.g
    compressjpeg.com    ]

  7. Lastly, minify all web-files(HTML, Css, JS, etc) for faster loading
