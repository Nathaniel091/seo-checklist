My SEO Checklist

  1. Favicon - (add an icon with the 'Link tag')
  2. Title tag
  3. META charset - (UTF-8)
  4. META Viewport - (remember to add 'META viewport' website scaling)
  5. META Author (Website author - important especially for blog websites)
  6. META description - (short description of the page's content which must include keywords)
  7. META keywords (Keyword must appear in title, meta description, heading, sub-heading, opening paragraph & anywhere possible)
  8. Add 'Alt' attribute for all images tags
  9. Add 'title' attribute for 'anchor' tags
  10. Mobile friendliness: mobile phone traffick is on the increase
  11. Compress images(JPEG) for faster loading [e.g compressjpeg.com]
  12. Lastly, minify all web-files(HTML, Css, JS, etc) for faster loading
